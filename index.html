How to Play

Rule 1: Each row can have only one occurence of all numbers from 1 - 9.

Rule 2: Each column can have only one occurence of all numbers from 1 - 9.

Rule 3: Each region of 9 cells bordered in orange can have only one occurence of all numbers from 1 - 9.

To start the game click 'Start' and a board will appear with a timer. You can use the mouse to select the number to place in a cell or use the keyboard.
